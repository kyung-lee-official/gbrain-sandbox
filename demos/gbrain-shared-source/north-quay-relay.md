# North Quay Relay

Fictional site lore for **cross-file retrieval** tests. Facts on this page alone are not enough to arm the relay.

## Location

The North Quay Relay sits on **Pier 7**. Its official callsign is **`ORION-LATCH`**.

## Arming rule (partial)

The relay arms only when an operator submits:

1. The callsign from this page
2. The duty officer's **color token** (see the duty roster page)
3. The **heptagon watch count** (see the heptagon watch page)

The arming formula is:

```text
{callsign}/{color-token}/{watch-count}
```

## What this page does not contain

- Color token
- Watch count
- Duty officer name
